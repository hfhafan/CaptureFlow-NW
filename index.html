Cocok dipakai untuk apa

Dokumentasi lapangan

Pilih target sekali, lalu setiap screenshot langsung masuk ke folder akhir yang sesuai.

Rekaman proses

Mulai rekam dari overlay dan simpan hasilnya langsung ke target aktif tanpa pindah folder manual.

Rapikan hasil dengan cepat

Kalau target tadi salah, file terakhir bisa langsung di-rename atau dipindah dari dalam app.

Cara pakai singkat

1. Siapkan proyek

Pilih folder root, pilih template, lalu buat folder utama agar semua target akhir siap dipakai.

2. Pilih target aktif

Gunakan layar proyek untuk memilih target akhir yang sedang dikerjakan, misalnya Before, After, atau Issue.

3. Mulai capture dari overlay

Nyalakan overlay, lalu tap Screenshot atau Mulai rekam. Hasil akan langsung disimpan ke target yang aktif.

4. Rapikan bila perlu

Buka Riwayat untuk rename, pindah, bagikan, buka file, atau buka folder hasil tangkapan.

Hal penting yang perlu diperhatikan

Beberapa app memang tidak mengizinkan tangkapan layar. Kalau layar tertentu tidak bisa ditangkap, itu biasanya karena perlindungan dari app tersebut.

Update aplikasi

Kalau ada versi baru, CaptureFlow NW akan meminta update. Tujuannya supaya alur capture, overlay, dan penyimpanan tetap stabil di berbagai versi Android.
